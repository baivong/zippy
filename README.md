# jQuery plugin ZippyShare

Zippyshare embed using jQuery.

## Quick use

```html
<div id="player"></div>
```

```js
// http://www26.zippyshare.com/v/XVcARUzL/file.html
$("#player").zippy({
    www: 26,
    file: "XVcARUzL",
    auto: true
});

```
